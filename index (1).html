🔔 Telegram Fire Alert Setup
When fire is detected, the system can send an instant Telegram message to your phone or group.
📋 SETUP STEPS
  1. Open Telegram → Search @BotFather
  2. Send /newbot → choose a name → copy Bot Token
  3. Search @userinfobot → send any message → copy your Chat ID
  4. Invite your bot to a group (optional) → use group Chat ID (starts with -)
  5. Enter both values in server.js → restart server
🌐 How to Get Your Device IP
📍 ESP32 IP ADDRESS
  1. Connect ESP32 to same WiFi as your PC
  2. Open Serial Monitor in Arduino IDE (115200 baud)
  3. Reset ESP32 → IP printed as: IP: 192.168.x.x
  4. Or check your router's admin page (usually 192.168.1.1)
  5. Enter IP in ESP32 Zone Devices above
💻 YOUR PC IP (for server.js)
  1. Press Win+R → type cmd → Enter
  2. Type ipconfig → look for IPv4 Address
  3. Use this IP in ESP32 sketch to send data to your server

📲 Install as Mobile App (PWA)FREE
FireGuard ICS can be installed on your phone or desktop as a native-like app — no app store needed.
📱 INSTALL STEPS
  1. Open FireGuard in Chrome browser on your phone
  2. Access via local IP: http://192.168.x.x:3000
  3. Tap ⋮ (3 dots) → "Add to Home Screen" or "Install App"
  4. Or tap the 📲 Install App button in the top bar when it appears
✅ Works offline  |  No browser UI  |  Professional app experience
🔌 ESP32 Connection GuideSTEP BY STEP
1
Upload Code to ESP32
Open fireguard_esp32.ino in Arduino IDE. Install libraries: DHT11, WiFi, ArduinoJson
2
Set Your WiFi Credentials
In the .ino file, update: const char* ssid = "YOUR_WIFI"; and const char* password = "YOUR_PASS";
3
Select Board & Upload
Tools → Board → ESP32 Dev Module  |  Tools → Port → select your COM port  |  Click Upload 🔝
4
Get ESP32 IP Address
Open Serial Monitor (115200 baud) → Reset ESP32 → Look for: IP: 192.168.x.x
5
Connect in Dashboard
Go to ESP32 Zone Devices card above → Enter IP next to Zone D → Click Connect Devices
6
✅ Live Data Active!
Zone D badge changes to ESP32 LIVE. Real sensor readings appear. When fire detected → Telegram + app notified instantly!
